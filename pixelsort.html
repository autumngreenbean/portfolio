# pixel-sort project

This project was made via java programming. To manufacture the featured effect, I expanded on a pixel-sorting code algorithim created by Kim Asendorf.

I updated the program to efficiently convert videos into large image sequences, process them, and encode them into .mp4 format using ffmpeg, rather than the original function of processing a single image.
